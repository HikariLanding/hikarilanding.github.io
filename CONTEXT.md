# CONTEXT

HikariLanding 官网的领域词汇表。

## 术语

### 品牌名片 (Brand Card)

本站的定位。一页式的组织门面，核心职责是传达 Hikari 的理念（warm, simple, quietly useful）与三条原则（Small tools, done well / Simple over clever / Light, not noise）。

**不是**作品集（Portfolio）：站点的价值不依赖项目数量，即使项目区为空也成立。

### 项目区 (Project Section)

品牌名片上预留的区域，展示所有「Hikari 项目」。设计为「会生长的」——随组织公开仓库的增加自动扩展，而非手工维护的静态清单。项目区为空时不是错误状态，显示「something new is being built」占位态。

### Hikari 项目 (Hikari Project)

出现在项目区的仓库，边界定义为：HikariLanding 组织下**公开、非 fork、非 archived** 的仓库，排除 `.github` 与官网仓库自身。按最近推送时间排序。fork 来的、已归档的、私有的仓库都**不是** Hikari 项目，即使它们属于该组织。
